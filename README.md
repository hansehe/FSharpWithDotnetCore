# Mixing F# & C#

- Simple try out on how to use F# with dotnet core.
- The solution is inspired by this tutorial:
    - https://docs.microsoft.com/en-us/dotnet/fsharp/get-started/get-started-command-line

## Run F# Console
- cd src/FApp
- dotnet run Hello World

## Run C# Console
- cd src/CApp
- dotnet run Hello World
